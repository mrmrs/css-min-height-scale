# css-min-height-scale

Functional CSS for min-height-scale

## Filesize

| File | Size |
|------|------|
| `dist/min-height-scale.css` | 1705 bytes |
| `dist/min-height-scale.min.css` | 1151 bytes (229 Gzipped) |

## Install

```sh
npm install css-min-height-scale
```

## Usage

### Import

```css
@import "css-min-height-scale";
```

### CDN

```html
<link rel="stylesheet" href="https://unpkg.com/css-min-height-scale/dist/min-height-scale.min.css">
```

### Direct

```html
<link rel="stylesheet" href="path/to/css-min-height-scale/dist/min-height-scale.min.css">
```

### Responsive

Responsive variants are available for each class with the following suffixes:

| Suffix | Media Query              |
|--------|--------------------------|
| `-s`   | `min-width: 32em`        |
| `-m`   | `min-width: 48em`        |
| `-l`   | `min-width: 96em`        |

Example: `.mn-hi1-m` applies the property at the medium breakpoint and above.

## Building

```sh
npm run build
```

Processes `src/min-height-scale.css` with [Lightning CSS](https://lightningcss.dev) and outputs to `dist/`.

- `dist/min-height-scale.css` — formatted
- `dist/min-height-scale.min.css` — minified
